# Code of Conduct

## Our commitment

We are committed to making participation in this project a harassment-free experience for everyone.

## Expected behavior

- Be respectful and constructive
- Focus on technical issues, not personal attacks
- Welcome different viewpoints and experiences

## Unacceptable behavior

- Harassment, insults, discrimination
- Publishing private information without consent
- Repeated bad-faith disruption

## Enforcement

Project maintainers may remove, edit, or reject comments, commits, code, and issues that violate this code.

## Reporting

Report incidents to maintainers via private channel or repository security contact.
